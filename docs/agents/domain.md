# Domain docs

OctoMeta is a single-context repository.

## Before exploring

- Read the root `CONTEXT.md` and use its canonical terms rather than synonyms listed under `_Avoid_`.
- Read the relevant system-wide decisions under `docs/adr/`.
- If an output would contradict an ADR, surface the conflict explicitly rather than silently overriding it.

`CONTEXT.md` is a glossary, not an implementation specification. ADRs record only hard-to-reverse decisions and their trade-offs.
