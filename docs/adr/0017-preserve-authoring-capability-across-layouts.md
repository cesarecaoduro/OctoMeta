# Preserve authoring capability across layouts

OctoMeta preserves every essential authoring task across compact, regular, and
expanded layouts instead of reducing phones and tablets to review-only experiences.
Presentation adapts to available space: compact layouts focus one workspace at a
time, larger layouts progressively compose workspaces, and secondary actions move
into discoverable menus rather than disappearing. This costs more design and
verification effort, but prevents device size or input method from changing what
someone is fundamentally allowed to do.
