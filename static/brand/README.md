# OctoMeta logo usage

Primary mark: octometa_mark_outline.svg
Filled mark: octometa_mark_filled.svg
App icon: octometa_icon_dark.svg
Wordmark: octometa_wordmark.svg
Horizontal lock-up: octometa_logo_horizontal_outline.svg

Colours
- Navy: #0B1020
- Indigo: #2B2E83
- Violet: #6C5CE7
- Lavender: #A99CFF
- Off-white: #F5F6F8

Rules
- Keep clear space around the mark equal to one terminal-node radius.
- Use the app icon without text.
- Use the outline mark at 48 px and above.
- Below 48 px, use the filled mark or simplified app icon.
- Do not distort, rotate, add shadows, or alter proportions.
- Test every use in light, dark, and monochrome contexts.
